![Language](https://img.shields.io/badge/language-Rust-orange.svg)

**Zapret Rust Interface** — это современный, быстрый и красивый графический/TUI интерфейс, разработанный на Rust, для управления утилитой **zapret** (winws) на основе популярного пака [Flowseal/zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube). 

*Забудьте про неудобные `.bat` файлы и консольное меню — управляйте обходом блокировок Discord, YouTube и Telegram в один клик.*

</div>

---

> [!WARNING]
> ### 🛡️ АНТИВИРУСЫ И ДРАЙВЕР WINDIVERT
> Компонент **WinDivert** (необходимый для перехвата и фильтрации трафика под Windows) может вызывать ложные срабатывания антивирусов (например, детект `Not-a-virus:RiskTool.Multi.WinDivert`). 
> * **Решение:** Добавьте папку с программой в исключения вашего антивируса или временно отключите проверку потенциально нежелательного ПО (PUA). Драйвер официально подписан для ядра Windows.

> [!IMPORTANT]
> Все исполняемые файлы в папке `bin` берутся напрямую из официальных релизов [bol-van/zapret](https://github.com/bol-van/zapret). Вы можете проверить их хэши вручную.

## ✨ Основные возможности

* 🚀 **Удобный запуск стратегий** — Визуальный выбор и мгновенное переключение между всеми доступными конфигурациями обхода (`ALT`, `FAKE`, `SIMPLE` и др.).
* ⚙️ **Управление службой в 1 клик** — Установка выбранной стратегии в автозапуск Windows (`services.msc`), удаление и проверка статуса службы без открытия командной строки.
* 🎮 **Умные фильтры трафика**:
  * **Game Filter** — Переключение обхода для игр (UDP/TCP выше 1023 порта) прямо из интерфейса.
  * **IPSet Filter** — Режимы `none`, `loaded` (только списки) и `any` (весь трафик).
* 🔍 **Встроенная диагностика** — Автоматическая проверка конфликтующих служб (GoodbyeDPI, Adguard, Killer Network, SmartByte), состояния службы фильтрации (BFE) и очистка кэша Discord.
* 🔄 **Автоматизация обновлений** — Обновление списков IPSet, конфигурации файлов `hosts` для веб-версий и проверка новых версий ПО.

## ⚙️ Предварительная настройка

1. **Включите Secure DNS (DNS over HTTPS):**
   * **Chrome / Edge:** Настройки ➔ Конфиденциальность и безопасность ➔ Безопасный DNS ➔ Выберите поставщика (например, Google или Квад9).
   * **Firefox:** Настройки ➔ Приватность и защита ➔ DNS через HTTPS ➔ Максимальная защита (рекомендуется указать `https://dns.google/dns-query`).
   * **Windows 11:** Рекомендуется включить DoH прямо в настройках сетевого адаптера ОС.
2. **Для роутеров Keenetic:** Включите опцию «Транзит запросов» в настройках роутера, чтобы избежать конфликтов DNS.

## 🚀 Установка и запуск

1. Скачайте актуальный релиз со страницы [Releases]().
2. Нажмите правой кнопкой мыши по архиву ➔ **Свойства** ➔ Поставьте галочку **«Разблокировать»** (если скачано через браузер).
3. Распакуйте архив в любую удобную папку. **Важно:** путь не должен содержать кириллицу, пробелы или спецсимволы (например, используйте `C:\zapretGUI\`).
4. Запустите приложение от **имени Администратора** (требуется для управления драйвером WinDivert и службами Windows).

## ℹ️ Краткое руководство по интерфейсу

### 1. Вкладка "Стратегии" (Strategies)
Позволяет протестировать обход в реальном времени. Если YouTube или Discord не работают на стандартной схеме, закройте текущую (иконка замочка в трее) и выберите другую (например, `FAKE TLS AUTO ALT`).

### 2. Вкладка "Служба" (Service Manager)
* **Install Service** — Прописывает выбранную рабочую стратегию в автозагрузку Windows.
* **Remove Services** — Полностью очищает систему от установленных служб обхода и драйвера WinDivert.
* **Check Status** — Показывает, запущен ли процесс `winws.exe` и активны ли драйверы.

### 3. Вкладка "Инструменты" (Tools)
* **Run Diagnostics** — Проверяет систему на наличие несовместимого софта и предлагает исправить проблемы.
* **Update Hosts** — Починка веб-версии Telegram и бесконечного подключения к голосовым каналам Discord.

## ☑️ Часто задаваемые вопросы (FAQ)

#### ❓ Что делать, если обход перестал работать?
Провайдеры могут изменять параметры фильтрации DPI. Если старая стратегия перестала работать, попробуйте альтернативные варианты (`ALT` / `FAKE`) или отредактируйте параметры вручную, используя документацию оригинального [zapret].

#### ❓ Конфликты с античитами (Faceit, Vanguard)
Некоторые античиты блокируют драйвер WinDivert. Перед запуском игры с античитом рекомендуется останавливать службу обхода, либо ознакомиться с инструкцией [windivert-hide](https://github.com/bol-van/zapret-win-bundle/tree/master/windivert-hide).

#### ❓ Требуется цифровая подпись драйвера для Windows 7?
Замените файлы `WinDivert.dll` и `WinDivert64.sys` в папке `bin` на совместимые версии из репозитория [zapret-win-bundle/win7](https://github.com/bol-van/zapret-win-bundle/tree/master/win7).

## 🗒️ Добавление собственных сайтов

Вы можете расширить списки ресурсов, отредактировав файлы в папке `lists/`:
* `list-general-user.txt` — для ваших доменов (поддомены учитываются автоматически).
* `list-exclude-user.txt` — домены-исключения.
* `ipset-all.txt` — для пользовательских IP-адресов и подсетей.
